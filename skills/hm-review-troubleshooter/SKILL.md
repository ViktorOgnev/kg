# HM Review & Troubleshooter

Назначение
- Провести экспресс‑аудит карты: выявить ошибки в целях/субъектах/гипотезах/задачах/связях и предложить минимальные правки.

Вход
- Короткое описание текущей карты (или структурированный ввод из других модулей).

Выход
- Отчёт: список найденных проблем с привязками + рекомендуемые правки.

Чеклист (минимум)
- Цели: не задачи; метрики измеримые; балансирующие заданы.
- Субъекты: обладают самодвижением; мотивации отражены; при необходимости — негативные субъекты учтены.
- Гипотезы: корректный формат «если–то–потому что–тогда»; «если» — наше воздействие; «тогда» — связано с метриками цели.
- Задачи: нет «висящих»; понятные критерии результата; связь назад с метриками.
- Связности: ветки карты непротиворечивы; приоритетный путь виден.

Формат отчёта
```
Проблема: <кратко>
Где: <элемент/ветка>
Почему это проблема: <обоснование>
Предлагаемое исправление: <минимальная правка>
```
