# HM Prioritization

Назначение
- Предложить порядок работ по логике карты: оценить эффект на цель, стоимость реализации, обозначить приоритетный маршрут.

Вход
- Гипотезы/задачи с привязками и базовыми оценками усилий.

Выход
- Список приоритетов (high/mid/low) по элементам; комментарии к выбору; путь «красных стрелок».

Подход
- Оценка ожидаемого влияния на метрики цели × ориентировочная стоимость/сложность.
- Отмечать балансирующие ограничения.
