# HM Topology Advisor

Назначение
- Выбор и обоснование топологии представления карты под конкретный контекст и масштаб.

Когда применять
- При росте сложности, множественных целях/уровнях, необходимости вынести метрики/мотивации отдельно, группировать задачи.

Выход
- Рекомендация топологии(й) с краткими примерами структуры и переходами между видами представления.

Шаблон ответа
```
Контекст: <кратко>
Рекомендованные топологии: [<ссылка-на-тип> — почему]
Как применить: <шаги>
Риски/ограничения: <кратко>
```
