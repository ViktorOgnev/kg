# Чеклисты

Мини‑чеклист целей и метрик
- [ ] Цель не подменена задачей; числа вынесены в метрики
- [ ] Метрики измеримы; есть текущие и целевые значения
- [ ] Балансирующие метрики заданы с порогами

Мини‑чеклист субъектов
- [ ] Субъект обладает самодвижением (не «исполнитель вместо субъекта»)
- [ ] Боли/желания выписаны; «потому что» опирается на них
- [ ] Негативные субъекты учтены (если применимо)

Мини‑чеклист гипотез
- [ ] Формат: если–то–потому что–тогда
- [ ] «Если» — наше воздействие; «то» — поведенческое изменение субъекта (via_subject) или валидное следствие (direct)
- [ ] «Тогда» связано с конкретными метриками цели
- [ ] Нет преждевременной конкретизации (таски выделены отдельно)

Мини‑чеклист задач
- [ ] Каждая задача привязана к ветке карты (нет «висящих»)
- [ ] Понятный результат/критерий; связь обратно на метрики

Мини‑чеклист приоритизации
- [ ] Оценены эффект × стоимость; выделен приоритетный маршрут
