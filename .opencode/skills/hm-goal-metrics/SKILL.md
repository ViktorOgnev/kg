# HM Goal & Metrics (OpenCode)

Что делать
- Переписать цель как желаемое состояние/результат, числа вынести в метрики.
- Описать метрики: current, target, type=leading|lagging.
- Добавить балансирующие метрики с порогами.

Форматы
- Текстовый список и/или JSON с полями goal, metrics[], balancing_metrics[].
